# PWA Icons

Dieses Verzeichnis enthält die Icons für die PWA-Funktionalität.

Um die Icons zu generieren, können Sie den mitgelieferten Icon-Generator verwenden:
1. Öffnen Sie die Datei `icon-generator.html` im Hauptverzeichnis des Projekts in einem Browser
2. Passen Sie die Hintergrundfarbe, Textfarbe und den Text an
3. Wählen Sie die gewünschte Größe aus den Buttons
4. Klicken Sie auf "Icon herunterladen"
5. Speichern Sie die heruntergeladene Datei in diesem Verzeichnis
6. Wiederholen Sie den Vorgang für alle benötigten Größen

Die folgenden Icon-Größen werden benötigt:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

Weitere Informationen finden Sie in der Datei `PWA-SETUP.md` im Hauptverzeichnis des Projekts.
